POST - Crear un nuevo vendedor
JSON:
{
    "name": "Juan Pérez",
    "email": "juan.perez@empresa.com",
    "phone": "+1234567890"
}
Casos de prueba:
Caso exitoso: Los 3 campos completos con datos válidos
Error validación: Falta campo "name"
Error validación: Email inválido ("juan@empresa")
Error validación: Teléfono muy corto ("123")
Error validación: Teléfono muy largo ("1234567890123456")
------------------------------------------------------------------------------------------------------------------------------------
